========================================
Документация для приложения Telegram Bot
========================================


Описание
========

Данное приложение реализует телелеграм-бота для целей тестирования. В приложении реализованы 
следующие функции:

- Приветствие пользователя и вывод перечня доступных команд.
- Запрос текущей погоды (температура, давление и влажность) в выбранном населенном пункте.
- Конвертация валют.
- Отправка случайной картинки с забавными животными.
- Создание опросов в групповых чатах.


Запуск тестового сервера
========================

Для запуска приложения в режиме тестирования выполните следующие действия:

1. Если на вашем ПК не установлен Git, то установите его в соответствии с инструкциями на сайте
   https://git-scm.com/downloads/.

2. Перейдите в рабочую папку (папка, в которую будет скопирован проект) и произведите клонирование
   проекта с GitHub. В командной строке выполните:

   git clone https://github.com/motuskov/telegram_bot.git

3. Если на вашем ПК не установлен Docker, то установите его в соответствии с инструкциями на сайте
   https://docs.docker.com/get-docker/.

4. Перейдите в папку проекта, который был клонирован на шаге 2 и в командной строке выполните:

   docker compose up -d

5. Дождитесь окончания развертывания и запуска контейнеров Docker.


Тестирование бота
=================

1. В приложении Telegram в строке поиска введите "#motuskov_test_bot" и выберите из списка позицию
   "Motuskov Test Bot".

2. Нажмите кнопку "START" и следуйте подсказкам.
